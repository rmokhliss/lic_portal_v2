SELECT-PX Design System

Une marque parente, des produits qui s'y accrochent.

Base réutilisable pour tous les produits monétique SELECT-PX. La marque parente porte l'identité (X chevron, dégradé cyan→bleu, typo) ; chaque produit s'identifie par un sigle court 3–4 lettres et un nom de projet. Drop-in pour Next.js 16 + Tailwind 4 + shadcn.

01 · FOUNDATIONS

Brand

L'identité parente est immuable : la tuile sombre arrondie, le chevron X au dégradé signature, et la wordmark SELECT-PX en surplomb. Tous les produits l'héritent. Seul le bloc produit (sigle + nom) varie.

Anatomy
A Tuile · constante. B Parent mark SELECT-PX · constant. C Slot produit · sigle + nom de projet (variable).
Règles
  • Sigle produit : 3–4 lettres majuscules · ex. LIC, PHS, MON, ACQ
  • Séparateur : underscore _ en gris 300 — jamais espace ni tiret
  • Suffixe métier facultatif : PORTAL, CONSOLE, OPS, DASH
  • Le X du parent ne change jamais de couleur — le dégradé est l'invariant
  • Pour différencier visuellement deux produits, n'utiliser que la couleur d'accent UI (voir §04), pas le logo
Voix

Sobre, technique, financier. Le ton d'un back-office bancaire — pas d'un SaaS marketing. Vocabulaire métier précis, pas de superlatifs, pas d'emoji. Les noms de produits restent descriptifs (Licensing Portal, Payment Hub Services) et jamais "fun".

15 · ENGINEERING

Install dans un nouveau produit

Trois étapes pour brancher le DS sur un nouveau produit SPX.

1 · Variables d'environnement

# .env.local — branding par produit NEXT_PUBLIC_PRODUCT_CODE=PHS NEXT_PUBLIC_PRODUCT_NAME="Payment Hub Services" NEXT_PUBLIC_PRODUCT_SUFFIX=CONSOLE

2 · Coller les tokens

Copier le bloc @theme §03 dans app/globals.css et importer les fonts via next/font/google.

3 · Installer les composants shadcn customisés

# Le DS fournit des recettes shadcn pré-thémées pnpm dlx shadcn@latest add button input badge table card dialog tabs sheet # puis appliquer les patches du DS (cf. /components/spx-overrides)

4 · Composant Brand

// components/Brand.tsx "use client"; import { brand } from "@/lib/brand"; export function BrandLockup({ size = 40 }: { size?: number }) { return ( <div className="flex items-center gap-3"> <SpxTile size={size} /> <div className="flex flex-col leading-tight"> <span className="font-mono text-[10px] tracking-[0.18em] uppercase text-gray-500">{brand.parent}</span> <span className="font-display font-extrabold text-lg"> {brand.code}<span className="text-gray-300 mx-0.5">_</span>{brand.suffix} </span> </div> </div> ); }
